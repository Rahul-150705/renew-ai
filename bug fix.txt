RENEW AI - BACKEND DEPLOYMENT & CACHING BUG FIX LOG
Date: 2026-05-24
========================================================================

1. CORS POLICY BLOCKING (FE-BE COMMUNICATION)
   - Issue: Vercel frontend was blocked by CORS preflight failures.
   - Cause: Multiple clashing CorsConfigurationSource beans in SecurityConfig and CorsConfig. @CrossOrigin annotations on controllers were also using wildcards which conflict with allowCredentials(true).
   - Fix: Consolidated all CORS logic into SecurityConfig.java. Removed all per-controller @CrossOrigin annotations. Enabled dynamic origin matching from ALLOWED_ORIGINS env var with a hardcoded fallback.

2. REDIS CACHE NOT WRITING (WRITES = 0)
   - Issue: Upstash Redis showed 0 writes despite @Cacheable annotations.
   - Cause A: Missing 'spring-boot-starter-cache' dependency in pom.xml.
   - Cause B: Missing '-parameters' compiler flag in pom.xml, preventing SpEL (#username) key resolution.
   - Cause C: 403 Forbidden errors preventing service method execution due to mismatch between controller @PreAuthorize and JWT authorities.
   - Fix: Added missing dependency, updated maven-compiler-plugin, and removed all role-based security requirements.

3. SSL CONNECTION TO UPSTASH
   - Issue: Redis connection failing or timing out.
   - Cause: Upstash requires TLS/SSL, but the app was defaulting to insecure connections.
   - Fix: Updated application.properties and .env to use REDIS_SSL=true and mapped the credentials correctly.

4. COLD START & STABILITY ON RENDER
   - Issue: Deployment was slow and instance would occasionally OOM.
   - Cause: Default JVM heap settings exceeded Render's 512MB limit.
   - Fix: Updated Dockerfile to use multi-stage builds (Eclipse Temurin JRE) and added -Xmx384m to limit memory usage.

5. GIT REPOSITORY CORRUPTION
   - Issue: Source code not appearing on Render/GitHub.
   - Cause: Nested .git folders and an accidental .gitignore entry blocking the /src folder.
   - Fix: Cleaned up git structure, fixed .gitignore, and performed a fresh push.
